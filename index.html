RECENT PROJECTS


Cube Field

An Android application that is created during nwHacks in a team of three. It allows the user to navigate through 3D space with randomly-generated cubes in a Virtual Reality simulation. This is made purely with Java along with our own implementation of perspective projection that converts a bunch of 3D vectors onto a 2D plane.


Programming Language: Java, Android

Mathematics: Linear Algebra, Perspective Projection



LetsChill

A web app to find a meetup location for two people in a local area. Given two locations, it will find options in between them and suggests cafes, restaurants, shopping center, theatres, etc for users. Build to provide convenient ways to plan group activities, meet-ups, hangouts, dates and family outings.


Frontend: HTML5, CSS3, Google Material Design, Javascript

API Used: Google AutoCompleteAPI, Google Places API, GoogleMaps API



MyCFL

A web application created during SportsHack in a team of three. It searches for a CFL athlete who is most similar to the user's attributes (e.g. height, weight, and location) and encourages new football fans to follow and displays the latest feeds from the athlete's social medias.


Frontend: HTML5, CSS3, Google Material Design, Javascript

Backend: Parse (Database), Python (Cosine Similarity for data analysis), Flask (API)



Moscrop Secondary School Official Application

Collaborated with a current student from Moscrop Secondary School to develop an iOS application using Swift that displays school's information, teachers' contacts as well as recent clubs and events updates. Improved user experience by adding different themes to allow users to customize their personal app interface.


Programming Language: Swift

Backend: Parse (Database)



Course Watcher

A MEAN-stack web application that serves the purpose of watching courses on UBC SSC, in which it informs users through emails when there is a seat open for a specific course. I adapted socket.io to implement real-time capability when adding and deleting new user requests.


Frontend: HTML5, CSS3, AngularJS

Backend: Node.js, Express.Js, MongoDB



Mind the Gap

An android application build in the CPSC 210 course. It displays the rail system in London along with real time data. It accurately pinpoints the location of each train stations with the integration of OsmDroid library.


Programming Language: Java

API Used: TransportforLondon API
